# SIEM Threat Detection Lab

## Overview
This project demonstrates a Microsoft Sentinel SIEM lab focused on threat detection, attack monitoring, and security event analysis in a Windows environment.

The lab simulates RDP brute-force attacks and enriches Windows Event Logs using PowerShell and external geolocation data sources.

## Technologies Used
- Microsoft Sentinel
- Azure Log Analytics
- PowerShell
- Windows Event Viewer
- Virtual Machines
- RDP Monitoring

## Features
- RDP brute-force attack detection
- Log enrichment with geolocation metadata
- Security event analysis
- Attack visualization dashboards
- Threat monitoring workflows

## Key Objectives
- Analyze Windows security logs
- Detect suspicious authentication attempts
- Improve visibility into attack sources
- Build SIEM monitoring and investigation skills
